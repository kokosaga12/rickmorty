# Rick and Morty Mini Dashboard

## Deskripsi Proyek
Rick and Morty Mini Dashboard adalah aplikasi React berbasis API publik yang menampilkan data karakter dari serial Rick and Morty. Aplikasi ini dikembangkan sebagai tugas Project Based Learning (PjBL) untuk mata kuliah Algoritma Pemrograman dan Struktur Data dengan topik Integrasi API Backend ke React.

## Fitur Utama
- Mengambil data dari API publik Rick and Morty
- Menggunakan 2 endpoint API berbeda:
  - Character API
  - Location API
- Menampilkan data dalam bentuk card/grid
- Fitur pencarian karakter
- Fitur filter berdasarkan status karakter
- Loading state
- Error handling
- Responsive design
- Menggunakan Tailwind CSS

## Teknologi yang Digunakan
- React JS
- Vite
- Axios
- Tailwind CSS
- Rick and Morty Public API

## API Endpoint
### Character Endpoint
```bash
https://rickandmortyapi.com/api/character
```

### Location Endpoint
```bash
https://rickandmortyapi.com/api/location
```
